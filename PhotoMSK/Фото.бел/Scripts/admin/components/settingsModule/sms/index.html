Внимание!

Стоимость SMS, настроенных через опцию «Свой шаблон», будет тарифицироваться по цене 1.9 ₽ за сообщение длиной до 69-ти символов.

Стоимость SMS, настроенных по одному из предложенных шаблонов, будет тарифицироваться по цене 1.5 ₽ за сообщение длиной до 69-ти символов.

Данные правила введены сотовыми операторами для регулирования рекламного трафика. Таким образом, все SMS, которые не соответствуют заранее согласованным шаблонам, будут приравниваться к рекламным и тарифицироваться по специальному тарифу.

Действительно только на территории РФ.
